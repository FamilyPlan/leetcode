给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串，判断字符串是否有效。

有效字符串需满足：

左括号必须用相同类型的右括号闭合。
左括号必须以正确的顺序闭合。
注意空字符串可被认为是有效字符串。

示例 1:
```quote
输入: "()"
输出: true
```
示例 2:
```quote
输入: "()[]{}"
输出: true
```
示例 3:
```quote
输入: "(]"
输出: false
```
示例 4:
```quote
输入: "([)]"
输出: false
```
示例 5:
```quote
输入: "{[]}"
输出: true
```
